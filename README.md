# 一、产品说明

### ⭐️ 1.加值宣言

1. **人脸识别甄别嫌疑犯**<br>
通过基于监控画面的图像为用户提高人脸识别的精确率功能。依托人脸识别算法和深度学习引擎，快速精准地定位人脸关键点等信息，将用户上传的照片打出特征点和人脸关键描述，使侦破案件不再只能依赖老警察的观察能力，也能够很便捷地呈现出嫌疑犯的形象的外貌特征。
2. **在线浏览关键信息**<br>
在人流量密集和案件高发的地区，基托人工的办法对嫌疑犯的甄别已经远远不能够满足于社会治安管理的需求，虽然原型还不够美观，但是已经做到了操作便捷和轻松交互，我们从功能设计入手倾斜向功能实现方面的对嫌疑犯照片的关键信息的在线浏览。
3. **肢体检测判别嫌疑犯行为**<br>
与传统的老警察线下耗费时间蹲点排查不同，使用肢体检测会更加容易辨别嫌疑犯在作案的细节动作，可以更好的兼顾到作案的方式方法的判断上，打造更有效的嫌疑犯排查方式。

综述：**安防放大镜**<br> 帮助警察、安保人员等进行嫌疑犯的追踪，皆在为提高安防原始照片放大后的人像清晰度，以提高对嫌疑犯的身份识别能力。

### ⭐️ 2.核心价值

- .对象为警察、安保人员等，拥有清晰度不足的照片并需要放大图片、进行人像抠图、图片清晰化，从而实现身份识别的需求的人员。

- .使用的方式简单，前后对比，一键完成。

### ⭐️ 3.用户痛点

视频监控最为明显的&quot;痛点&quot;是——看不清嫌疑犯的样子。看清，就是核心价值。

import json

def Face_attribute_detection():

        "parameter": {
            "rsp_media_type": "jpg",
        },
        "extra": {},
        "media_info_list": [{
            "media_data": image_64_encode,
